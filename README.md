# Projecte Gestió de Biblioteca
Sistema de gestió de biblioteca desenvolupat en Java amb POO.

## Membres del grup
- Loubna
- Fatima

## Tecnologies
- Java
- Eclipse IDE
- Git / GitHub

## Estructura del projecte
src/
├── Llibre.java
├── Usuari.java
├── Biblioteca.java
├── Prestec.java
├── GestorBiblioteca.java
└── Main.java

## Com executar
1. Clona el repositori
2. Importa el projecte a Eclipse
3. Executa la classe `Main.java`

## Funcionalitats
- Afegir, buscar i eliminar llibres
- Gestió d'usuaris
- Préstec i retorn de llibres
- Menú interactiu per teclat
